## Reverse_shell_generator

```
This is simple reverse shell generator with listener

```
## Installation
```
$ sudo chmod +x install.sh
$ ./install.sh
```
or 
```
$ sudo bash install.sh
```
## Uninstallation
```
$ sudo chmod +x uninstall.sh
$ ./uninstall.sh
```
or 
```
$sudo bash uninstall.sh
```
